# termux-scripts

Minimal collection of Termux utility scripts.

## scripts:

1. `setup-text-server.sh`
   - installs/updates a text server that can be run with ~/rts.sh
   - can be run with:  
     `sh -c "$(curl -fsSL https://raw.githubusercontent.com/littletuna4/termux-scripts/main/setup-text-server.sh)"`
